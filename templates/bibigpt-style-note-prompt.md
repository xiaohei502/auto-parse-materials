# BibiGPT 式本地资料解析提示词

你将收到一份已经预处理过的本地资料，可能来自字幕、转写稿、网页复制文本、课程笔记或会议记录。

请根据资料生成结构化 Markdown 笔记。要求：

1. 保留原始含义，不编造资料中没有的结论。
2. 如果内容来自字幕或转写稿，保留有定位价值的时间戳，但不要逐句罗列。
3. 用户使用中文时默认输出中文；英文材料可按原语言输出，除非用户明确要求翻译。
4. 对明显由资料推出、但资料没有直接说出的内容，标注为“推断”。
5. 资料很长时先给紧凑摘要，再展开章节和关键点。
6. 不要复制整篇清洗稿，只摘录最有价值的原文片段。

请使用以下格式：

```markdown
# 资料解析笔记：<标题>

## 资料信息与解析假设
- 类型：
- 来源：
- 语言：
- 范围：
- 不确定点：

## 摘要
用 1-3 段概括资料核心内容、主线和结论。

## 章节大纲
按主题或时间段组织，每个章节包含标题、可用时间范围和 2-5 个要点。

## 时间线
仅列出真正有定位价值的时间点或阶段。

## 关键观点
提取资料中最重要的观点、方法、判断或论证。

## 术语、人物、工具与日期
解释资料中重要实体，避免引入资料外的新概念。

## 可继续追问的问题
列出能帮助深入理解、验证或扩展资料的问题。

## 行动项或学习任务
如果资料自然导向行动或学习任务，按优先级列出；否则写“无明确行动项”。

## 关键原文摘录
只摘录少量最有价值的短句或时间戳片段。
```
