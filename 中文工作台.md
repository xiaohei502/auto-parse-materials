# 本地资料解析中文工作台

这份文件是日常入口。底层文件夹保留英文名称，避免脚本、插件和跨平台工具出问题；你实际使用时按下面的中文入口理解即可。

## 快速入口

| 中文入口 | 实际目录 | 用途 |
|---|---|---|
| 入口：待处理资料 | `inputs/` | 放字幕、转写稿、网页正文、会议记录、课程材料 |
| 存档：原文记录 | `records/originals/` | 保存用户粘贴的原始文本或网页正文 |
| 存档：转写稿 | `records/transcripts/` | 保存字幕、逐字稿、口播稿、转写稿 |
| 存档：预处理稿 | `records/preprocessed/` | 保存脚本清洗后的中间稿 |
| 出口：最终笔记 | `outputs/` | 保存普通总结、详细页、最终 Markdown 笔记 |
| 出口：二次产出 | `artifacts/` | 保存高光、公众号图文、改写文章、AI 对话 |
| 知识库 | `library/` | 管理总结记录、收藏、待整理、待办、闪念 |
| 合集 | `collections/` | 按课程、主题、项目组织资料 |
| 探索 | `explore/` | 手动收集订阅源和热门资料 |
| 模板 | `templates/` | 普通总结、详细页、高光、公众号图文等提示词 |
| 说明 | `manual/` | 工作流操作说明 |

## 推荐对话入口

直接把资料发给 Codex 时：

```text
使用 auto-parse-materials 解析下面资料，保存原文记录和转写稿，再问我生成哪种产出。
```

如果已经知道目标：

```text
使用 auto-parse-materials 解析下面资料，保存原文记录和转写稿，直接生成详细页 + 高光笔记。
```

## 标准处理流

```text
入口：待处理资料
  -> 存档：原文记录 / 转写稿
  -> 存档：预处理稿
  -> 出口：最终笔记
  -> 出口：二次产出
  -> 知识库 / 合集
```

## 产出类型

- 普通总结：适合快速理解。
- 详细页：适合系统阅读和长期保存。
- 高光笔记：适合回看、剪辑、复盘和传播片段。
- 原文细读：适合保留逐字稿、润色稿和书面改写。
- 公众号图文：适合把视频或转写稿改成图文草稿。
- 课程学习笔记：适合课程、教材、PPT、转写课稿和复习资料。
- 自定义总结：适合按你的具体目标生成。
